CSS Cascading and Inheritance

如果多条规则都适用于同一个元素并产生冲突，优先级最高的规则会被使用。

如果没有适用的规则，则会使用 默认样式#。

不同规则类型之间的优先级

根据 样式的来源# 和是否具有 !important# 标记，样式会有多种类型，它们的优先级从高到底为：

  1. transition origin
  2. important user agent origin
  3. important user origin
  4. important author origin
  5. animation origin
  6. normal author origin
  7. normal user origin
  8. normal user agent origin

同一类规则之间的优先级

  1. specificity# 越高，优先级越高。
  2. specificity 相同时，定义的越晚，优先级越高。
